## Code Structure

This project is a flat-structure static website.

### Files & Folders

- **`/index.html`**: The home page, featuring a terminal-style typing effect.
- **`/about.html`**: About Leo.
- **`/projects.html`**: Showcase of work and contributions.
- **`/contact.html`**: Contact information and social links.
- **`/styles.css`**: Styling css file
- **`/script.js`**: Client side interactive logic.
- **`/locales/`**: Directory containing translation data.
  - `en.json`: English language strings.
  - `zh.json`: Chinese language strings.


## How to Run

Since this is a static website, you can simply open `index.html` in any modern web browser (Internet Explorer is **NOT** one of them!)
